# st-jorink
My st config with some patches.

## Installing st-jorink

`git clone https://github.com/MJorink/st-jorink.git`

`cd st-jorink`

`sudo make clean install`

## Configuration

The configuration of dwm is done by editing config.h
and (re)compiling the source code. (sudo make clean install)
